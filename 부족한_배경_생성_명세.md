# 부족한 배경/CG 생성 명세

작성일: 2026-06-03

기준 파일: `../도파민때문에_시나리오_proto.xlsx`

현재 `story-data-excel.js`와 `config.js` 기준으로는 배경 파일 누락이 0개다. 다만 proto 엑셀에는 앞으로 반영될 새 배경명과 CG가 더 들어 있으며, 일부는 기존 범용 배경으로 대체하기에는 장면 전달력이 부족하다. 아래 목록은 다른 에이전트에게 생성을 맡기기 위한 제작 명세다.

## 공통 스타일

- 비율: 16:9 가로형. 가능하면 기존 생성 에셋과 맞춰 `1672x941` 또는 최소 `1280x720`.
- 장르: 한국 대학생 연애 비주얼노벨.
- 스타일: 기존 캐릭터 스프라이트와 기존 CG에 맞춘 애니풍 비주얼노벨 일러스트.
- 색감: 선명하지만 과하게 채도가 높지 않은 부드러운 톤. 기존 `assets/bg`의 밝은 애니 배경들과 어울리게.
- 금지: 워터마크, 글자, 말풍선, UI 오버레이, 로고, 읽을 수 있는 실제 브랜드명.
- 대화창이 하단에 깔리므로, 하단 20~25%는 주요 얼굴/핵심 행동을 피해서 구성.
- CG는 캐릭터가 배경 이미지 안에 포함된 완성 장면이다. CG가 표시되는 동안 개별 캐릭터 스프라이트는 띄우지 않는 원칙을 지킨다.
- 주인공은 명확한 얼굴 노출을 피하고, 필요하면 뒷모습/옆모습/손/어깨 정도로 처리한다.
- 수진 기준 외형: 긴 검은 머리, 파란 눈, 분홍 케이블 니트와 리본, 흰 치마. 기존 `assets/char/수진_Normal.png`, `assets/char/수진_Surprised.png`를 강하게 참고.
- 건호/혜지 포함 컷은 기존 캐릭터 스프라이트와 복장/머리색/분위기 일관성을 맞춘다.

## 생성 후 연결 작업 메모

생성된 파일은 `도파민때문에/assets/bg/`에 저장한다.

그 다음 `config.js`의 `ASSET_MANIFEST.backgrounds`에 proto 엑셀의 배경명을 그대로 키로 추가한다. 예:

```js
"(CG) 백스테이지에 있는 주인공과 수진": "./assets/bg/backstage_protagonist_sujin_cg.png",
"문 세트": "./assets/bg/stage_door_set.png",
"카톡방 화면": "./assets/bg/kakaotalk_group_chat_screen.png",
```

CG 배경은 시나리오 데이터에서 표시 직후 `clear characters`가 오도록 유지한다. 일반 배경으로 돌아갈 때는 명시적으로 새 `background` 노드를 넣는다.

야외로 나가거나 밤길처럼 천천히 열리는 장면은 현재 규칙상 `fadeSlide`와 `slideSpeed: 0.7`을 사용한다. 같은 장소 안의 CG 컷인/시점 전환은 보통 `fadeSlide`, 완전히 장소가 바뀌는 경우는 `fadeBlack`을 사용한다.
