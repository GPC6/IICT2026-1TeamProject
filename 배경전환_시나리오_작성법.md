# 배경 전환 효과 시나리오 작성법

활성 시나리오 파일은 `story-data-excel.js`입니다. 배경 효과는 `type: "background"` 노드에 옵션을 추가해서 설정합니다.

## 기본값

옵션을 아무것도 쓰지 않으면 기본 전환은 `fadeBlack`입니다.

```js
{
  "type": "background",
  "name": "MT 장소"
}
```

## 짧은 문자열 방식

가장 간단한 방식입니다.

```js
{
  "type": "background",
  "name": "MT 장소",
  "transition": "fadeSlide"
}
```

사용 가능한 값:

- `fadeBlack`: 기존 배경에서 검은 화면으로 페이드아웃 후 새 배경으로 페이드인
- `fadeSlide`: 검은 화면이 된 뒤 새 배경이 한쪽부터 투명도가 자연스럽게 번지며 페이드인
- `none`: 전환 효과 없이 즉시 변경

`cut`, `instant`도 `none`과 같습니다. `slide`, `fade-slide`도 `fadeSlide`와 같습니다.

## 세부 옵션 방식

전환 시간이나 슬라이드 방향까지 조절할 때는 객체로 씁니다.

```js
{
  "type": "background",
  "name": "MT 장소",
  "transition": {
    "type": "fadeSlide",
    "duration": 700,
    "direction": "right",
    "slideDuration": 420
  }
}
```

옵션 의미:

- `type`: `fadeBlack`, `fadeSlide`, `none`
- `duration`: 전환 시간(ms). 120부터 2000 사이 숫자만 사용
- `direction`: `fadeSlide`에서만 의미 있음. `right`는 오른쪽부터, `left`는 왼쪽부터 새 배경의 투명도가 번지듯 나타남
- `slideDuration`: `fadeSlide`에서 새 배경이 열리는 시간(ms). 숫자가 클수록 더 천천히 열림
- `slideSpeed`: `fadeSlide`에서 새 배경이 열리는 속도 배율. `1`이 기본, `2`는 2배 빠름, `0.5`는 절반 속도

`slideDuration`과 `slideSpeed`를 둘 다 쓰면 `slideDuration`이 우선 적용됩니다.

```js
{
  "type": "background",
  "name": "MT 장소",
  "transition": {
    "type": "fadeSlide",
    "duration": 700,
    "direction": "right",
    "slideSpeed": 1.6
  }
}
```

## 납작한 옵션 방식

객체를 쓰기 번거로우면 아래처럼 따로 적어도 됩니다.

```js
{
  "type": "background",
  "name": "MT 장소",
  "transition": "fadeSlide",
  "transitionDuration": 700,
  "transitionDirection": "right",
  "transitionSlideDuration": 420
}
```

## 추천 사용 기준

- 장소가 완전히 바뀔 때: `fadeBlack`
- 같은 장소 안에서 시점만 바뀌거나 장면을 부드럽게 열고 싶을 때: `fadeSlide`
- 아주 빠르게 컷 전환하고 싶을 때: `none`

배경 전환이 진행되는 동안에는 대사창과 선택지가 잠시 멈춰서 표시되지 않습니다.
