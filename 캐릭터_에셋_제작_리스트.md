# 캐릭터 에셋 제작 리스트

이 문서는 `story-data-excel.js`, `도파민때문에_시나리오.xlsx`, `도파민때문에_시나리오_follow.xlsx`를 기준으로 본게임에 필요한 캐릭터 이미지 에셋을 정리한 제작 목록이다.

현재 게임 구조상 캐릭터 스탠딩 이미지는 `config.js`의 `ASSET_MANIFEST.characters`에 캐릭터명과 감정명을 등록해 사용한다. 파일은 PNG, 캐릭터 외 영역은 투명 배경으로 제작한다.

## 공통 제작 기준

- 파일 형식: PNG
- 배경: 완전 투명
- 캔버스 기준: 현재 정규화 에셋과 같은 `512 x 1536` 권장
- 캐릭터 실제 높이: 기존 `assets/char_balanced` 기준과 맞추기
- 파일명 규칙 권장: `캐릭터명_감정명.png`
- 감정명은 가능하면 JS에 이미 쓰인 이름과 맞추기: `Normal`, `Happy`, `Surprised`, `Flustered`, `Frightened`, `Focused`, `hopeful`, `Sad`

## 2순위: 수진 장면별 의상/상태 변형

수진은 주연이라 같은 감정이어도 장면별 맥락이 다르다. 제작 여유가 있다면 감정만 바꾸는 것보다 의상과 상태를 분리하는 편이 몰입도가 높다.

| 장면 | 필요한 변형 | 설명 | 우선도 |
| --- | --- | --- | --- |
| EP1 편의점 | 편의점 알바 또는 밤 근무 캐주얼 | 졸다가 계산대를 보는 첫 만남 장면. `sleepy`, `Surprised`, `Flustered`와 잘 맞음 | 높음 |
| EP2 동아리 OT | 캠퍼스 기본 의상 | 첫 동아리 만남과 자기소개 장면. 현재 기본 스탠딩의 기준 의상으로 사용 가능 | 높음 |
| EP3 MT | MT 캐주얼 | 술자리, 랜덤게임, 친구들과 편하게 있는 장면. 약간 풀어진 분위기 필요 | 중간 |
| EP4-A 공포방탈출 | 외출복 + 겁먹은 상태 | 공포방탈출 입구/내부. `Frightened`, `Flustered`, `Focused` 중심 | 높음 |
| EP4-B 파스타집/길거리 | 데이트 느낌 외출복 | 주인공과 단둘이 만나는 루트. `Normal`, `Focused`, `Happy`, `Flustered` 중심 | 높음 |
| EP5 동아리방 | 동아리방 캐주얼 | 미팅 사건 대화. 당황/해명/어색함이 중요 | 중간 |
| EP6 백스테이지 | 공연 스태프/작업 상태 | 공구박스, 드라이버, 무대 수리 장면. `Focused`, `Flustered`, `Happy` 중심 | 높음 |
| 엔딩 밤길 | 공연 후 밤길 상태 | 고백 성공/거절 모두 사용. `hopeful`, `Happy`, `Sad`, `Surprised` 중심 | 높음 |

## 3순위: 특수 CG / 컷인 에셋

아래는 스탠딩 이미지보다 장면 전달력이 중요한 CG 성격의 이미지다. 배경 이미지로 등록하거나 별도 CG 표시 기능을 추가해 사용할 수 있다.

| 장면 | 필요한 CG | 설명 | 우선도 |
| --- | --- | --- | --- |
| EP1 첫만남 | 졸고 있는 수진 | 편의점에서 수진이 고개를 꾸벅꾸벅 떨구는 장면 | 높음 |
| EP1 첫만남 | 화들짝 깨는 수진 | 계산 요청에 놀라 깨는 장면 | 높음 |
| EP3 MT | MT 장소에 둘러앉은 네 명 | 주인공, 수진, 건호, 혜지가 랜덤게임을 시작하는 단체 CG | 중간 |
| EP3 MT | 소주병을 향해 뻗는 수진의 손 | 수진이 벌칙주를 마시려는 클로즈업 | 높음 |
| EP3 MT | 수진의 손을 잡는 건호의 손 | 건호가 잔을 대신 가져가는 클로즈업. 관계 긴장감이 생기는 장면 | 높음 |
| EP4-A 공포방탈출 | 천장에서 귀신이 떨어지고 놀라는 수진 | 공포방탈출 핵심 이벤트 CG | 높음 |
| EP4-B 둘이 | 파스타집의 주인공과 수진 | 단둘이 만나는 데이트 루트 진입 CG | 중간 |
| EP4-B 둘이 | 파스타집을 나와 걸어가는 두 사람 | 길거리 대화와 관계 진전 장면 | 중간 |
| EP6 정기공연 | 수진이 나사를 조이는 장면 | 손이 떨리지만 참고 무대 세트를 고치는 장면 | 높음 |
| EP6 정기공연 | 수진과 주인공 하이파이브 | 문제가 해결되고 마주보고 웃는 장면 | 중간 |
| 엔딩 | 고백 성공 수진 클로즈업 | “나도 너 좋아해”에 맞는 설렘/부끄러움 컷 | 높음 |
| 엔딩 | 고백 거절 수진 클로즈업 | 미안함과 조심스러운 거절을 표현하는 컷 | 높음 |

## 현재 구현상 추가로 필요한 작업

### config.js 캐릭터 등록

스탠딩 에셋을 제작한 뒤 `config.js`의 `ASSET_MANIFEST.characters`에 감정별 파일 경로를 추가해야 한다.

예시:

```js
수진: {
  default: "./assets/char_balanced/수진.png",
  Normal: "./assets/char/수진_Normal.png",
  Happy: "./assets/char/수진_Happy.png",
  Surprised: "./assets/char/수진_Surprised.png",
  Flustered: "./assets/char/수진_Flustered.png",
  Frightened: "./assets/char/수진_Frightened.png",
  Focused: "./assets/char/수진_Focused.png",
  hopeful: "./assets/char/수진_hopeful.png",
  Sad: "./assets/char/수진_Sad.png"
}
```

### 선배1 등록

`story-data-excel.js`에는 `선배1`이 등장하지만 현재 기본 캐릭터 에셋에는 없다. 백스테이지 장면을 제대로 보이게 하려면 `선배1` 캐릭터를 새로 제작하고 `config.js`에 등록해야 한다.

### 감정명 통일

현재 감정명은 대부분 첫 글자 대문자이지만 `hopeful`만 소문자로 되어 있다. 코드를 크게 바꾸지 않으려면 에셋 등록도 `hopeful` 그대로 맞추는 것이 안전하다.
