# 도파민때문에

**도파민때문에**는 선택형 스토리와 미니게임이 섞인 비주얼노벨 게임입니다.

플레이어는 주인공이 되어 일상 속 대화를 따라가고, 순간마다 어떤 말을 할지 선택합니다. 선택은 인물과의 관계뿐 아니라 도파민 수치에도 영향을 주며, 중간에 등장하는 미니게임 결과까지 더해져 마지막 엔딩이 달라집니다.

## 게임 소개

이 게임은 "자극을 무조건 많이 얻는 것이 정말 좋은가?"라는 질문에서 출발합니다.

대화 속 선택지는 설렘, 회피, 충동, 조절 사이에서 플레이어를 흔들고, 미니게임은 도파민을 높이거나 낮추는 감각을 직접 플레이로 보여줍니다. 중요한 것은 점수를 끝없이 올리는 것이 아니라, 지금의 상태를 읽고 적당한 균형을 찾아가는 일입니다.

## 플레이 방식

- 대사를 읽고 선택지를 고르며 이야기를 진행합니다.
- 선택에 따라 도파민과 호감도 수치가 달라집니다.
- 스토리 중간의 미니게임 결과가 이후 흐름에 반영됩니다.
- 누적된 선택과 수치에 따라 서로 다른 엔딩을 볼 수 있습니다.

## 핵심 재미

**도파민때문에**는 빠른 자극과 느린 관계 사이에서 균형을 찾는 게임입니다.

어떤 선택은 당장은 즐겁지만 수치를 불안정하게 만들 수 있고, 어떤 선택은 느리지만 관계를 더 단단하게 만들 수 있습니다. 플레이어는 이야기와 미니게임을 오가며 자신만의 리듬으로 결말에 도달하게 됩니다.

## 실행

`index.html`을 브라우저에서 열면 게임을 플레이할 수 있습니다.
