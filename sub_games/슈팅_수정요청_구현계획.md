## 3. 설계 결정

### 3.1 도파민과 강화의 역할

도파민은 동시에 세 가지 의미를 가진다.

| 역할 | 설명 |
| --- | --- |
| 위험도 | `90` 이상이면 과열 타이머가 차며, 5초 지속 시 패배 |
| 강화 자원 | 우클릭 또는 `Shift/X`로 패시브 강화를 구매할 때 소비 |
| 플레이 템포 | 도파민이 높으면 플레이어 이동 기본 속도와 적 압박이 약간 상승 |

도파민을 많이 모으면 강해지지만, 너무 오래 들고 있으면 패배한다. 따라서 플레이어는 적 처치와 아이템으로 도파민을 모으고, 적절한 시점에 강화로 소비해야 한다.

### 3.2 패시브 강화 목록

그라디우스의 6단 파워업을 그대로 복제하지 않고, 현재 게임에 맞춰 아래 6개를 사용한다.

| 단계 | 내부값 | 이름 | 필요 도파민 | 효과 |
| --- | --- | --- | --- | --- |
| 1 | `speed` | 속도 증가 | `20` | 이동 속도 보너스 `+0.7` |
| 2 | `double` | 더블 샷 | `35` | 기본 탄환을 위/아래 2발로 발사 |
| 3 | `bomb` | 하단 폭탄 | `50` | 공격 시 아래로 떨어지는 폭탄 추가 |
| 4 | `laser` | 관통 레이저 | `65` | 기본 탄환이 관통형 레이저가 됨 |
| 5 | `option` | 보조 펫 | `80` | 플레이어를 따라다니는 보조 발사체 1개 생성 |
| 6 | `shield` | 실드 | `90` | 실드 `2` 부여, 이미 실드가 있으면 최대 `4`까지 보강 |

강화는 한 번 구매하면 지속된다. 이미 보유한 강화는 다시 구매하지 않는다. 구매 가능한 강화 중 가장 높은 단계가 아니라, 낮은 단계부터 순서대로 구매한다. 즉 `speed`를 사야 `double`을 살 수 있다.

### 3.3 도파민 보상 밸런스

발사 도파민 증가는 제거한다. 적 처치 보상은 적 종류와 무관하게 `+3`으로 고정한다.

아이템은 적 처치 시 `14%` 확률로 생성되며, 획득 시 도파민 `+8`을 준다. 게이트는 웨이브 생성 시 `4%` 확률로만 등장한다.

이 기준에서 예상 흐름은 다음과 같다.

| 상황 | 예상 도파민 변화 |
| --- | --- |
| 적 10마리 처치 | `+30` |
| 아이템 1개 획득 | `+8` |
| `speed` 구매 | `-20` |
| `double` 구매 | `-35` |
| 과열 회피용 `shield` 구매 | `-90`, 실드 확보 |

플레이어가 계속 처치만 하면 90을 넘지만, 강화를 구매하면 도파민이 내려간다. 이 구조가 "도파민 조절하며 생존"의 핵심 루프다.

```js
useSkill() {
  const nextUpgrade = this.getNextUpgrade();
  if (!nextUpgrade) {
    this.resultText = "모든 강화 활성화";
    return;
  }

  if (this.dopamine < nextUpgrade.cost) {
    this.resultText = `${nextUpgrade.name}: 도파민 ${nextUpgrade.cost} 필요`;
    return;
  }

  this.addDopamine(-nextUpgrade.cost);
  this.activateUpgrade(nextUpgrade.key);
  this.resultText = `${nextUpgrade.name} 활성화: 도파민 -${nextUpgrade.cost}`;
}

getNextUpgrade() {
  return this.getUpgradeCatalog().find((upgrade) => !this.upgrades[upgrade.key]) || null;
}

activateUpgrade(key) {
  this.upgrades[key] = true;
  if (key === "shield") this.player.shield = Math.min(4, this.player.shield + 2);
  if (key === "option") {
    this.optionDrone = { x: this.player.x - 42, y: this.player.y, trail: [] };
  }
}
```
