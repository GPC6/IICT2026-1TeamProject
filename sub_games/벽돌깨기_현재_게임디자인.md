# 도파민 벽돌깨기 현재 게임 디자인

작성 기준 파일: `sub_games/breakblocks.js`

## 1. 역할

도파민 벽돌깨기는 본게임 중간에 실행되는 짧은 턴제 미니게임이다.

```js
new BrickBreakerGame(initialDopamine, { maxTurns: 6 })
```

본게임에서 현재 도파민을 받아 시작하고, 종료 시 `getDopamine()`으로 최종 도파민을 돌려준다. 내부 도파민 범위는 `0~100`이다.
첫 본게임 벽돌깨기는 스토리 옵션에서 `maxTurns: 5`로 더 짧게 진행된다.

## 2. 기본값

| 항목 | 값 |
| --- | --- |
| 게임 영역 | `980 x 640` |
| 컬럼 수 | `9` |
| 기본 최대 턴 | `6`턴 |
| 턴 옵션 범위 | `1~20`턴 |
| 초기 블럭 줄 | `3`줄 |
| 바닥선 | `y = 560` |

## 3. 목표와 조작

목표는 점수가 아니라 6턴 동안 도파민을 너무 낮거나 높게 망치지 않고 끝내는 것이다.

| 입력 | 동작 |
| --- | --- |
| 마우스 이동 | 조준 방향 결정 |
| 좌클릭 | 공 발사 |
| 게임오버 후 클릭/Enter/R | 본게임으로 복귀 |

첫 공이 바닥에 닿은 위치가 다음 턴 발사 위치가 된다. 이후 공들은 그 위치로 슬라이딩된 뒤 사라진다.

한 턴이 오래 지속되면 진행 보정이 걸린다.

| 조건 | 효과 |
| --- | --- |
| 발사 후 10초 경과 | 게임속도 `2배속` 적용 및 화면 표시 |
| 발사 후 15초 경과 | 남은 공을 자동으로 아래로 떨어뜨려 턴을 마무리 |

## 4. 공 발사

공 개수는 도파민에 따라 정해진다.

```js
Math.max(1, Math.min(10, Math.floor(this.dopamine / 10) + this.permanentBallBonus))
```

도파민 40~50으로 시작하면 보통 `4~5`개가 발사된다.

## 5. 블럭

한 줄은 9칸 중 8칸에 블럭이 생성되고, 남은 1칸은 비거나 아이템이 들어간다.

| 종류 | 내부값 | 효과 |
| --- | --- | --- |
| 자극 블럭 | `stim` | 맞을 때 도파민 `+1` |
| 완화 블럭 | `recover` | 파괴 시 도파민 `-8` |
| 일반 블럭 | `normal` | 도파민 변화 없음 |

블럭에는 효과값이 아니라 남은 내구도만 표시한다.

턴이 지날수록 새로 생성되는 블럭의 HP가 조금 오르지만, 현재 기본값에서는 최대 `+2`까지만 오른다.

## 6. 아이템

아이템은 새 줄의 빈칸에 간헐적으로 생성된다. 공이 아이템에 닿으면 즉시 효과가 적용되고, 공/블럭 색으로 효과가 보인다.

| 아이템 | 효과 |
| --- | --- |
| 완화 큐브 | 이번 턴 자극 블럭 도파민 증가 차단 |
| 고자극 큐브 | 이번 턴 자극 증가량 상승 |
| 강화 큐브 | 이번 턴 블럭 피해량 상승 |
| 관통 큐브 | 이번 턴 공이 블럭을 관통 |
| 예열 큐브 | 이후 발사 공 수 영구 `+1` |
| 안정화 큐브 | 도파민을 안정 구간 쪽으로 이동 |
| 재흡수 큐브 | 이번 턴 완화 블럭 효과 상승 |

## 7. 종료 조건

| 조건 | 결과 |
| --- | --- |
| `maxTurns` 생존 | 생존 완료 |
| 블럭이 바닥까지 내려옴 | 그 시점 현재 도파민으로 종료 |

블럭이 바닥까지 내려와도 도파민을 `0`이나 `100`으로 강제 변경하지 않는다. 플레이 결과는 종료 순간의 도파민이다.
도파민이 높아도 과열로 강제 종료되지는 않으며, 대신 높은 도파민 구간에서는 완화 블럭 비중이 늘어난다.

## 8. 튜토리얼

본게임에서 별도 튜토리얼 옵션이 없으면 기본 튜토리얼이 표시된다.

1. 6턴 동안 진행되고 점수가 아니라 도파민이 결과라는 점
2. 마우스로 조준하고 블럭 숫자는 내구도라는 점
3. 블럭이 아래까지 내려오면 현재 도파민으로 종료된다는 점

## 9. 현재 디자인 의도

- 플레이 시간이 짧아야 본게임 흐름을 덜 끊는다.
- 첫 본게임 벽돌깨기는 5턴으로 낮춰 규칙 파악을 쉽게 한다.
- 실패해도 현재 도파민을 반환해 과한 벌점 느낌을 줄인다.
- 상세 설명은 미니게임 안의 긴 HUD보다 파미니 튜토리얼로 전달한다.
