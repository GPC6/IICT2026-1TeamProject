# 뉴럴 사이드 슈터 현재 게임 디자인

작성 기준 파일: `sub_games/shooting.js`

## 1. 역할

뉴럴 사이드 슈터는 본게임 중간에 실행되는 짧은 회피/공격 미니게임이다.

```js
new SideShooterGame(initialDopamine, { durationSeconds: 45, difficulty: 0 })
```

본게임에서 현재 도파민을 받아 시작하고, 종료 시 `getDopamine()`으로 최종 도파민을 돌려준다. 내부 도파민 범위는 `0~100`이다.

## 2. 기본값

| 항목 | 값 |
| --- | --- |
| 게임 영역 | `980 x 620` |
| 기본 플레이 시간 | `45`초 |
| 플레이 시간 옵션 범위 | `10~120`초 |
| 난이도 옵션 | `0~10`, 적 생성 시 HP 보너스 |
| 초기 라이프 | `3` |

## 3. 조작

교수님 피드백에 따라 기본 조작은 마우스만으로 이해되게 정리했다.

| 입력 | 동작 |
| --- | --- |
| 마우스 이동 | 플레이어가 마우스 위치를 따라 이동 |
| 좌클릭 | 공격, 꾹 누르면 빠른 자동 발사 |
| 우클릭 | 현재 파워 칸 기술 사용 |
| 방향키/WASD | 보조 이동 입력 |
| Space/Shift/X | 보조 공격/기술 입력 |

HUD에는 마우스 조작 중심 문구만 표시한다.

## 4. 도파민 흐름

| 상황 | 도파민 변화 |
| --- | --- |
| 시간 경과 | 도파민이 높을수록 더 크게 지속 감소 |
| 적 처치 | `+3` |
| 자극 캡슐 | `+8` |
| 안정 캡슐 | `-10` |
| 피격 | `+10` |
| 흡수 필터 사용 | `-25` |
| 수용체 게이트 | `55` 쪽으로 보정 |
| 라이프 소진 | 현재 도파민으로 종료 |

라이프가 0이 되어도 도파민을 0으로 강제 변경하지 않는다. 마지막 피격으로 변한 현재 값이 그대로 결과가 된다.
도파민이 높아도 과열로 강제 종료되지는 않는다.

색상 규칙은 벽돌깨기와 맞춘다.

| 색 | 의미 |
| --- | --- |
| 빨강/분홍 | 도파민 상승, 자극, 피격, 적 처치 |
| 초록/민트 | 도파민 감소, 안정 캡슐, 흡수 필터 |
| 노랑 | 파워 캡슐, 기술 선택 |

## 5. 파워 미터

파워 캡슐을 먹으면 `powerLevel`이 1칸 오른다. 현재 칸의 기술은 우클릭으로 사용한다.

| 칸 | 내부값 | 이름 | 효과 |
| --- | --- | --- | --- |
| 1 | `absorb` | 흡수 필터 | 도파민 `-25`, 반복 사용 가능 |
| 2 | `double` | 더블 샷 | 위/아래 2발 발사 |
| 3 | `speed` | 속도 증가 | 이동 속도 크게 증가, 도파민 지속 감소 보조 |
| 4 | `laser` | 관통 레이저 | 기본 탄환이 관통 |
| 5 | `option` | 보조 펫 | 추가 탄환 발사 |
| 6 | `shield` | 실드 | 실드 2 부여 |

흡수 필터는 패시브 보유 기술이 아니라 즉시 사용 기술이며, 여러 번 사용할 수 있다. 흡수 필터를 제외한 기술은 사용 후 파워 미터에서 빈칸으로 남고, 빈칸에 도착할 수는 있지만 선택하거나 사용할 수 없다. 현재 위치가 빈칸이면 노란 테두리와 대각선 표시로 커서 위치를 보여준다. 기술이 발동되면 캐릭터 주변에 색 빛 효과가 표시된다.

## 6. 적과 난이도

적은 타입별 묶음 웨이브로 등장한다.

| 대열 | 구성 |
| --- | --- |
| `droneLine` | `drone` 5기 |
| `shooterWedge` | `shooter` 3기 |
| `tankOscillate` | `tank` 2기 |

플레이테스트 피드백에 따라 기본 HP, 스폰 압박, 탄막 빈도는 이전보다 낮췄다.

| 타입 | 기본 HP | 특징 |
| --- | --- | --- |
| `drone` | `1` + 보정 | 기본 적 |
| `shooter` | `2` + 보정 | 탄환 발사 |
| `tank` | `4` + 보정 | 느리고 단단함 |

`difficulty` 옵션은 새로 생성되는 적의 HP에 더해진다.

## 7. 종료 조건

| 조건 | 결과 |
| --- | --- |
| 설정 시간 생존 | 생존 성공 |
| 라이프 0 | 현재 도파민으로 종료 |

## 8. 튜토리얼

본게임에서 별도 튜토리얼 옵션이 없으면 기본 튜토리얼이 표시된다.

1. 45초 생존, 마우스 이동, 좌클릭 홀드 자동 발사
2. 빨강은 도파민 상승, 초록은 도파민 감소라는 색상 규칙
3. 적 처치는 모두 도파민 `+3`, P 캡슐은 우클릭 기술 선택
4. 속도 증가와 흡수 필터의 도파민 감소 효과, 기술 발동 시 색 빛 효과

## 9. 현재 디자인 의도

- 키보드를 몰라도 마우스만으로 플레이 가능해야 한다.
- 미니게임은 45초 안에 끝나 본게임 흐름을 덜 끊어야 한다.
- 실패해도 현재 도파민을 반환해 과한 벌점을 줄인다.
- 적 체력과 스폰 압박은 플레이테스트용으로 쉽게 둔다.
