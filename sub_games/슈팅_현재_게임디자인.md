# 뉴럴 사이드 슈터 현재 게임 디자인

작성 기준 파일: `sub_games/shooting.js`

이 문서는 현재 코드에 구현된 슈팅 미니게임의 실제 동작을 정리한 것이다. 앞으로 수정할 때는 이 문서를 기준으로 변경할 규칙을 지정한다.

## 1. 게임 역할

뉴럴 사이드 슈터는 본게임 중간에 실행되는 미니게임이다.

본게임에서 현재 도파민 값을 받아 시작하고, 미니게임 종료 시 최종 도파민 값을 본게임에 돌려준다.

```js
new SideShooterGame(initialDopamine, { durationSeconds: 90, difficulty: 0 })
```

종료 후 본게임은 아래 메서드로 도파민 값을 가져간다.

```js
getDopamine()
```

게임 내부 도파민은 `0~100`으로 제한된다.

## 2. 본게임 연동

`config.js`의 `SUB_GAME_MANIFEST`에서 `sideShooter`로 등록되어 있다.

| 항목 | 값 |
| --- | --- |
| 미니게임 ID | `sideShooter` |
| 표시 제목 | `뉴럴 사이드 슈터` |
| 클래스명 | `SideShooterGame` |
| BGM | `game2` |

## 3. 기본 화면 크기와 초기 상태

| 항목 | 값 |
| --- | --- |
| 게임 영역 너비 | `980` |
| 게임 영역 높이 | `620` |
| 기본 플레이 시간 | `90`초 |
| 플레이 시간 옵션 범위 | `5~180`초 |
| 난이도 옵션 | `0~10`, 적 생성 시 체력 보너스 |
| 초기 라이프 | `3` |
| 도파민 상한 | `100` |

`resetGame()`에서 플레이어, 탄환, 적, 게이트, 아이템, 강화 상태를 초기화한다.

초기 플레이어 상태는 다음과 같다.

```js
{ x: 110, y: this.h / 2, r: 18, shield: 0 }
```

## 4. 승리와 패배

| 조건 | 결과 |
| --- | --- |
| 설정된 시간만큼 생존 | 생존 성공 |
| 도파민 `90` 이상 상태가 5초 지속 | `도파민 과열` |
| 라이프 `0` 이하 | `라이프 소진`, 결과 도파민 `0` |

과열 중에는 HUD에 남은 과열 시간이 표시된다.

## 5. 조작

| 입력 | 동작 |
| --- | --- |
| `WASD` 또는 방향키 | 이동 |
| `Space` 또는 좌클릭 | 기본 공격 |
| `Shift`, `X`, 또는 우클릭 | 다음 패시브 강화 구매 |

발사 시 도파민은 증가하지 않는다.

## 6. 도파민 흐름

| 상황 | 도파민 변화 |
| --- | --- |
| 시간 경과 | 매 프레임 `-0.012` |
| 적 처치 | `+1` |
| 자극 캡슐 획득 | `+8` |
| 안정 캡슐 획득 | `-10` |
| 피격 | `+10` |
| 흡수 필터 사용 | `-15` |
| 수용체 게이트 통과 | `55` 쪽으로 보정 |
| 라이프 소진 | `0`으로 고정 |

적 종류별 처치 효과 차이는 없다. 모든 적은 처치 시 동일하게 도파민 `+1`을 준다.

## 7. 파워 미터와 패시브 강화 시스템

강화는 그라디우스식 파워 미터를 참고한 선택형 방식이다. 적 처치 시 낮은 확률로 나오는 파워 캡슐을 먹으면 `powerLevel`이 1칸 오른다.

`powerLevel`은 `0~6` 사이에서 움직인다. `6`칸이 찬 뒤 파워 캡슐을 더 먹어도 `6`에 머문다.

현재 선택된 칸의 기술은 `Shift`, `X`, 우클릭으로 획득한다. 기술 획득 시 `powerLevel`은 `0`으로 리셋된다.

이전 단계 기술을 가지고 있지 않아도 현재 선택된 칸의 기술을 바로 획득할 수 있다.

| 순서 | 내부값 | 이름 | 비용 | 효과 |
| --- | --- | --- | --- | --- |
| 1 | `speed` | 속도 증가 | 파워 1칸 | 이동 속도 `+0.7` |
| 2 | `double` | 더블 샷 | 파워 2칸 | 위/아래 2발 발사 |
| 3 | `absorb` | 흡수 필터 | 파워 3칸 | 도파민 `-15`, 반복 사용 가능 |
| 4 | `laser` | 관통 레이저 | 파워 4칸 | 기본 탄환이 관통 레이저가 됨 |
| 5 | `option` | 보조 펫 | 파워 5칸 | 플레이어를 따라다니며 추가 탄환 발사 |
| 6 | `shield` | 실드 | 파워 6칸 | 실드 2 부여, 최대 4 |

이미 보유한 패시브 기술 칸을 선택하면 새로 획득하지 않고 `이미 보유 중` 메시지가 표시된다. 이 경우 파워 미터는 유지된다.

`흡수 필터`는 패시브 보유 기술이 아니라 즉시 사용 기술이다. 파워 3칸에서 사용하면 도파민을 `15` 낮추고 파워 미터가 `0`으로 리셋된다. 이후 다시 파워 3칸을 모으면 같은 기술을 다시 사용할 수 있다.

## 8. 기본 공격과 탄환

강화 상태에 따라 공격 구성이 바뀐다.

| 상태 | 발사 |
| --- | --- |
| 기본 | 전방 일반탄 1발 |
| 더블 샷 | 위/아래 일반탄 2발 |
| 관통 레이저 | 기본탄 대신 관통 레이저 |
| 보조 펫 | 플레이어 공격 시 보조 펫도 추가 발사 |

| 탄환 종류 | 피해 | 관통 |
| --- | --- | --- |
| 일반탄 | `1` | 없음 |
| 레이저 | `2` | 있음. 단, 같은 탄환은 같은 적을 한 번만 타격 |

## 9. 적 대열

적은 단일 랜덤 생성이 아니라 웨이브 대열로 등장한다.

현재 대열 템플릿은 타입별 묶음 단위로 나뉜다.

| 대열 | 구성 |
| --- | --- |
| `droneLine` | `drone` 5기가 줄지어 등장 |
| `shooterWedge` | `shooter` 3기가 벌어진 대형으로 등장 |
| `tankOscillate` | `tank` 2기가 위아래로 요동치며 등장 |

적이 화면 왼쪽으로 나가면 제거되지 않고 오른쪽에서 다시 들어온다. 처치된 적만 제거된다.

시간이 지날수록 웨이브 압박이 증가한다.

| 난이도 요소 | 변화 |
| --- | --- |
| 웨이브 생성 간격 | 시간 경과에 따라 점점 짧아짐 |
| 웨이브 구성 | 후반부에는 같은 타입의 추가 적이 붙음 |
| 적 속도 | 시간 경과와 도파민 수치에 따라 증가 |
| 적 체력 | 생성 옵션 `difficulty`와 후반 보정에 따라 증가, `tank`는 더 많이 증가 |
| shooter 탄환 간격 | 시간 경과와 높은 도파민에서 짧아짐 |

## 10. 적 타입

적 타입은 처치 효과 차이가 아니라 행동/대열 역할을 나누기 위해 사용한다.

| 타입 | 표시 | 체력 | 특징 |
| --- | --- | --- | --- |
| `drone` | 마름모 | `4` + 후반 보정 | 기본 적, 일렬 대열 |
| `shooter` | 원형 코어 + 포신 | `5` + 후반 보정 | 탄환을 발사하는 유일한 타입, 벌어진 대형 |
| `tank` | 큰 사각형 | `8` + 후반 보정 | 느리고 단단함, 위아래로 요동 |

적 탄환은 `shooter` 타입만 발사한다.

## 11. 아이템과 게이트

적 처치 시 `22%` 확률로 아이템이 생성된다.

| 요소 | 효과 |
| --- | --- |
| 파워 캡슐 | 파워 미터 `+1` |
| 자극 캡슐 | 도파민 `+8` |
| 안정 캡슐 | 도파민 `-10` |
| 수용체 게이트 | 도파민을 `55` 쪽으로 보정 |

수용체 게이트는 웨이브 생성 시 `4%` 확률로만 등장한다.

## 12. HUD

HUD에는 다음 정보가 표시된다.

| 표시 | 내용 |
| --- | --- |
| 제목 | `뉴럴 사이드 슈터` |
| 조작 안내 | `WASD/방향키 이동, Space/좌클릭 공격, Shift/X/우클릭 강화` |
| 도파민 미터 | 현재 도파민 |
| 파워 미터 | 현재 선택된 기술 칸과 보유 강화 수 |
| 과열 타이머 | 도파민 90 이상일 때 남은 시간 |
| 라이프/남은 시간 | 현재 생존 상태 |

우클릭은 강화 입력으로 사용되며, 캔버스에서는 브라우저 기본 우클릭 메뉴가 뜨지 않도록 막는다.

기술 획득 시 플레이어 주변 원형 펄스는 표시하지 않는다. 대신 파워 미터에서 방금 획득한 기술 칸이 짧게 반짝인다.

## 13. 현재 디자인의 주요 압력

1. 적을 처치하면 도파민이 아주 조금 오른다.
2. 파워 캡슐을 먹어야 기술 선택 칸이 오른다.
3. 현재 선택된 기술은 이전 기술 없이도 바로 얻을 수 있다.
4. 도파민은 강화 비용이 아니라 위험/컨디션 게이지로 남는다.
5. 낮은 도파민에서는 이동과 탄속이 약해지고, 높은 도파민에서는 플레이어도 강해지지만 적 속도/탄막/웨이브 압박도 강해진다.
6. 90 이상을 오래 유지하면 패배하므로 자극/안정 캡슐 선택이 중요하다.
7. 적은 타입별 묶음 대열로 계속 누적되며 후반부로 갈수록 체력과 수가 늘어난다.
8. `shooter` 타입만 탄환을 쏘기 때문에 우선순위 타겟이 된다.
9. 점수는 사용하지 않으며 최종 산출물은 종료 시 도파민 수치다.
10. 생성 시 `difficulty`를 지정하면 모든 적의 체력 레벨을 올릴 수 있다.
